# 一、node.js 学习

# nodemon 包

  在调试 Node.js 项目的时候，如果修改了代码，则要频繁的手动 close 掉，然后重新启动，非常繁琐。

  而使用 nodemon 工具，它可以监听项目文件的变动，并自动帮我们重启项目。

# 1、安装
npm install nodemon -g

# 2、使用
# 原来：
node app.js
# 现在：
nodemon app.js

# 二、性能优化 —— base64

<body>
  <!-- 普通的方式： -->
  <img src="./picture.jpg" />
  <!-- base64的方式： -->
  <img src="data:image/jpg;base64,/……" />
</body>

优点：

  • base64，拿到 html 页面的同时就拿到了图片，不用单独的再次发送 ajax 请求。

缺点：

  • 数据体积变大

应用范围：

  • 小图片，小 logo 图片
