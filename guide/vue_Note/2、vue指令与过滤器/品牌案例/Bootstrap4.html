# Bootstrap4

官方介绍

Bootstrap 4 (opens new window) 是全球最受欢迎的前端开源工具库，它支持 Sass 变量和 mixin、响应式栅格系统、自带大量组件和众多强大的 JavaScript 插件。

简单的说：

  • 预处理器由 less 切换到 sass
  • 响应式布局方式改为：rem + Flexbox 布局
  • 移动端设备优先并且适配了移动端
  • 依赖 jQuery

# 1、内边距（pading）外边距（margin）

① m 代表 margin

  • .m-1 相当于 margin: 0.25rem !important;
  • .m-2 相当于 margin: 0.5rem !important;
  • .m-3 相当于 margin: 1rem !important;

  针对单条边

  • .mt-3、.ml-3、.mr-3、.mb-3
  • .mx-3、.my-3

② p 代表 padding

  • .p-1 相当于 padding: 0.25rem !important;
  • .p-2 相当于 padding: 0.5rem !important;
  • .p-3 相当于 padding: 1rem !important;

  针对单条边

  • .pt-3、.pl-3、.pr-3、.pb-3
  • .px-3、.py-3

# 2、display 和 float

d 代表 display

  • .d-inline-block 行内块元素
  • .d-block 块级元素

float 值

  • .float-left
  • .float-right
  • .float-none

# 3、尺寸大小

w 代表 width

  • .w-25 相当于 width: 25%
  • .w-50 相当于 width: 50%
  • .w-75 相当于 width: 75%
  • .w-100 相当于 width: 100%

区间

  • .mw-100
  • .mh-100
