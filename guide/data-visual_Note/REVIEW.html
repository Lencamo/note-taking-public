# 一、sort()方法

array.sort(compareFunction)
1

描述：

  用原地算法对数组的元素进行排序，并返回数组

使用：

compareFn(a, b) 🎉 返回值 排序顺序
> 0 a 在 b 后
< 0 a 在 b 前
=== 0 保持 a 和 b 的顺序

# 代码示例

a - b 表示按 升序 排列（即：从小到大）

  • 数组内的数字排序
const numbers2 = [4, 2, 5, 1, 3]
numbers2.sort((a, b) => a - b)

console.log(numbers2)
// [1, 2, 3, 4, 5]

# 二、定时器

# 知识回顾

回调函数：

  回调，就是回头调用的意思。即在完成一定的事情后，再回头调用这个函数。而普通函数是按照代码顺序直接调用的。

  需要注意的是，下面内部的函数也是回调函数：

  • element.onclick = function() {}
  • element.addEventListener('click',fn)

同时，也要注意 this 的指向问题

# 1、setTimeout()

var timeoutID = scope.setTimeout(function[, delay, arg1, arg2, ...]);

  设置一个定时器，该定时器在定时器到期后执行一个函数或指定的一段代码。

# 2、setInterval()

var intervalID = setInterval(func, [delay, arg1, arg2, ...]);
var intervalID = setInterval(function[, delay]);
var intervalID = setInterval(code, [delay]);
1
2
3

描述：

  Window 和 Worker 接口提供的 setInterval() 方法重复调用一个函数或执行一个代码片段，在每次调用之间具有固定的时间间隔。

代码演示：

var intervalID = setInterval(myCallback, 500, 'Parameter 1', 'Parameter 2')

function myCallback(a, b) {
  // Your code here
  // Parameters are purely optional.
  console.log(a)
  console.log(b)
}
