# this 指向

  this 的指向是当我们调用函数的时候确定的，调用方式的不同决定了 this 的指向不同。

调用方式 this 指向 描述
普通函数 window
构造函数 实例对象 (原型对象里的方法 指向的是 实例对象)
对象方法 改方法所属对象
事件绑定方法 绑定事件对象
定时器函数 window
立即执行函数 window

（一般指向的是 🚩 调用者）

// 1、普通函数
function fn() {
  console.log(this) // Window { …… }
}
// window.fn()
fn()

// 2、构造函数
function Star(name, age) {
  this.name = name
  this.age = age

  console.log(this) // Star { name: '鹿先生',age: 22}
}
var that
Star.prototype.sing = function () {
  that = this
  console.log(this) // Star { name: '鹿先生',age: 22}
}
var lencamo = new Star('鹿先生', 22)
lencamo.sing()

console.log(lencamo) // Star { name: '鹿先生',age: 22}
console.log(that === lencamo) // true

// 3、对象方法
var obj = {
  say: function () {
    console.log(this) // {say: f()} 其实就是 obj
  }
}
obj.say()

// 4、事件绑定方法
var btn = document.querySelector('button')
btn.onclick = function () {
  console.log(this) // 打印了button整个元素
}

// 5、定时器函数
// window.setTimeout(function () {
setTimeout(function () {
  console.log(this) // Window { …… } 🤔
}, 3000)

// 6、立即执行函数
// (function () {
//   console.log(this)
// })()

# 一、call()方法

function.call(thisArg, arg1, arg2, ...)
1

描述：

  使用一个指定的 this 值和单独给出的一个或多个参数来调用一个函数。简单的来说，即：

  • 可以调用函数
  • 可以改变函数的 this 指向

# 代码演示

  • 调用函数
function fn() {
  console.log('函数被调用了')
  console.log(this) // Window { …… }
}

// fn()
fn.call()

# 二、apply()方法

function.apply(thisArg, [argsArray])
1

描述：

  这个函数的语法与 call() 几乎相同，但根本区别在于，call() 接受一个参数列表，而 apply() 接受一个参数的单数组。

# 应用场景

  • 数组合并
// 1、传统的push()方法可接受可变数量的参数
var sports = ['soccer', 'baseball']
var total = sports.push('football', 'swimming')
1
2
3
// 2、现在的apply()方式可接受数组
var vegetables = ['parsnip', 'potato']
var moreVegs = ['celery', 'beetroot']

// 将第二个数组融合进第一个数组
// 相当于 vegetables.push('celery', 'beetroot');
Array.prototype.push.apply(vegetables, moreVegs)

console.log(vegetables)
// ['parsnip', 'potato', 'celery', 'beetroot']

# 二、bind() 方法

function.bind(thisArg[, arg1[, arg2[, ...]]])
1

描述：

返回一个新的函数 ，在 bind() 被调用时，这个新函数的 this 被指定为 bind() 的第一个参数，而其余参数将作为新函数的参数，供调用时使用。简单的来说，即：

  • 会返回一个原函数的拷贝，并拥有指定的 this 值和初始参数
  • 不会调用原函数
  • 可以改变原函数内部的 this 指向

特点 ✨：

在不调用函数的情况下，改变 this 指向

# 应用场景

  • 定时器函数中的使用
// 1、原始方案
btn.onclick = function () {
  this.disabled = true
  var that = true

  setTimeout(function () {
    that.disabled = false
  }, 3000)
}

// 2、高级写法
btn.onclick = function () {
  this.disabled = true

  setTimeout(
    function () {
      that.disabled = false
    }.bind(this),
    3000
  )
}
