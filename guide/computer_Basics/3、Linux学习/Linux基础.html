# 一、Linux 认知

# 1、文件权限

r ：可读 read

w ：可写 write

x ：可执行 excuse

# 2、基础命令

reboot # 重启
shutdown # 关机

ifconfig # 查看网络接口属性

netstat -lntp # 正在监听的端口🎉

npm install --unsafe-perm --registry=https://registry.npm.taobao.org/ # npm的v7.0.0版本已解决--unsafe-perm问题，无需指定

ps -aux | grep java # 查看进程（包含进程号）

kill -9 [进程号] # 关闭某个进程

tail -f sys-info.log # 监听日志

nohup java -jar ruoyi.jar & # 部署jar包（后台运行）

# 二、切换目录、查看、查找、返回命令 💖

# 1、切换目录

cd dir
1

cd ~ 切换到 home 目录

cd .. 切换到上一级目录

cd - 返回上次使用的目录

   . . /表示同一等级目录； . /表示当前目录 ； ~/ 表示 home 目录 （即：~/ 与 /home/lencamo/ 等效）； / 表示 根目录。

# 2、显示目录文件

# 1、普通
ls
ls -F  # 显示是文件、还是目录

# 2、详细
ll     # 是ls -l的别名👀
ls -lh # 显示内容大小（详细）
ls -lt # 显示按时间排序（详细）

# 3、全部
ls -al # 显示隐藏文件

# 五、压缩包

  -z：打包同时压缩 ✨

-c：产生.tar 打包文件

-x：解压.tar 文件

-t：查看压缩包内容

  -v：显示详细信息 verbose

  -f：指定压缩后的文件名 file

# 1、.rar

# 解压：
rar xvf FileName.rar
# 压缩：
rar cvf FileName.rar DirName
