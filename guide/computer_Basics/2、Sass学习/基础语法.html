# 3、数据类型

  其实到现在我们已经可以感受到 sass 是一种类编程语言。所以它相较于 css，可以支持更多的数据类型（maps、数据、空值、布尔值等等）。

// 低级
$layer-index: 10;
$border-width: 3px;
$font-base-family: 'Open Sans','Helvetica','Sans-Serif';
$top-bg-color: rgba(255,147,29,0.6);
$block-base-padding: 6px 10px 6px 10px;
// 高级
$blank-mode: false;
$var: null;
$color-map: (color1: #fa0000,color2:#fbe200,color3:#95d7eb);
$fonts:(serif: 'Helvetica Neue',monospace: 'Consolas');

.container {
  // 常规类型
  $font-size: 16px !global; // 设置全局变量
  font-size: $font-size;

  // 布尔值
  @if $blank-mode {
    background-color: #000;
  }
  @else {
    background-color: #fff;
  }

  // null类型
  content: type-of($var);
  content: length($var);

  // map类型
  color: map-get($color-map, color3 );
}

.footer {
  font-size: $font-size;
}

.wrap {
  // map类型（key值不存在时，生成的css会忽略该空值）
  font: 18px bold map-get($fonts, 'sans');
}

# 二、@import 导入

# 1、导入方式

// 正确方式（会从中引入sass文件内容）
@import 'public';

// 错误方式（将做普通css语法，不会编译或者说引入sass文件内容）
@import 'public.css';
@import url(public);
@import 'http://xxx.com/xxx';
@import 'landscape' screen and (orientation: landscape);

# 2、主题文件

  有一类 scss 文件，内容仅定义了样式变量供其他文件引用 —— 主题文件。如果还让它生成目标 css 文件，显然是没有必要的。

① 定义：

  将样式文件的名字以_开头即可，这样的文件不会被编译为 css 文件（如：_theme.scss、_shop.scss、_user.scss 等模块化样式文件）。

② 注意事项：

_public.scss和public.scss文件在引入时，会被看做一个文件

@import 'public'; // 相当于引入了💖_public.scss文件
1

# 3、拓展

  在项目中，如果我们想要使用@别名，需要在前面加上一个**~**才可以

.scrollbar-wrapper {
  background: url('~@/assets/common/leftnavBg.png') no-repeat 0 100%;
}

# 三、混合指令 ✨

  其实关于 mixin 混入，在 vue、小程序、uniapp 中的使用并不少见 😂。

使用场景：

  很多地方要用到，但要根据不同的场景进行灵活使用的样式就可以使用 mixin 混入。

# 1、选择器角度

@mixin block {
  width: 96%;
  margin-left: 2%;
  border-radius: 8px;
  border: 1px #f6f6f6 solid;
}

.container {
  @include block;
}

/* ------ */

@mixin warning-text {
  .warn-text {
    font-size: 12px;
    color: rgb(255,234,155);
    line-height: 180%;
  }
}

.box {
  @include warning-text;
}

# 2、传参角度 👀

① 参数

  一般情况下，变量作为参数时最好设置默认值。

@mixin block-padding($top: 0, $right: 0, $botton: 0, $left: 0) {
  padding-top: $top;
  padding-right: $right;
  padding-bottom: $botton;
  padding-left: $left;
}

.container {
  @include block-padding($left: 20px, $right: 20px);
}

# 四、继承

  关于继承，最广泛的应用就是组件库中的按钮（如：border-radius、font-size 等样式）。

# 1、继承示例

.alert {
  padding: 15px;
  margin-bottom: 20px;
  border: 1px solid transparent;
  border-radius: 4px;
  font-size: 12px;
}

.alert-info {
  @extend .alert;
  color: #31708f;
  background-color: #d9edf7;
  border-color: #bce8f1;
}

.alert-success {
  @extend .alert;

  background-color: #dff0d8;
